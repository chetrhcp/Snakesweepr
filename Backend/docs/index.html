Backend API Documentation

Runner/Manager

Server

Used to initialize and setup all aspects of the server. Configurations for the web and socket server are set and then the rest of the handlers are initialized.

  • There are no functions associated with this module.

Interface Handlers

Site Handler

Handles all connections and data pathways dealing with the website. This website serves data in a semi-formatted, raw format, that allows admin users to see the status of the server and all the functions that are being utilized.

  • [public] init(app,infoRouter) | Server Application Instance, Info Router Object

Socket Handler

Handles all connections and data pathways dealing with WebSockets. The WebSocket server deals with data sent from the website (in terms of semi-live data) and the app (all data is handled via sockets from the app).

  • [public] init(app,infoRouter) | Server Application Instance, Info Router Object

Routing

Info Router

  • [public] init()
  • [public] handle_data(data,conn) | Raw Data From Web/Socket Server, Connection Identifier (in this case, the generated socketid).
  • [private] middleman(dataPacket) | DataPacket object. Ideally created from the submitted data in handle_data().
